= Where do I report security bugs found in this theme? =

Please report security bugs found in the source code of the undefined plugin through the [Patchstack Vulnerability Disclosure  Program](https://patchstack.com/database/vdp/ad4f0b73-2c90-4835-af1a-21a933838077). The Patchstack team will assist you with verification, CVE assignment, and notify the developers of this plugin.
